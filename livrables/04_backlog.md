# Backlog

## US01 - Alléger l'interface

En tant qu'utilisateur du tableau de bord, je souhaite une vue plus sobre qui n'affiche que les informations essentielles, afin de réduire le nombre d'éléments chargés et d'améliorer la lisibilité.

Bonne pratique associée : **Garder une interface lisible**.

## US02 - Réduction du rafraîchissement automatique

En tant qu'utilisateur régulier, je souhaite que la page ne se recharge pas automatiquement toutes les quelques secondes, afin d'éviter des appels réseau répétés et inutiles.

Bonne pratique associée : **Limiter les rafraîchissements automatiques**.

## US03 - Chargement initial léger

En tant qu'utilisateur du tableau de bord, je souhaite que la page charge uniquement les données utiles au premier affichage, afin que l'ouverture soit plus légère.

Bonne pratique associée : **Charger seulement le nécessaire**.

## US04 - Pagination et détail à la demande

En tant qu'utilisateur qui consulte les dossiers, je souhaite que la liste soit paginée et que les notes et l'historique se chargent à l'ouverture d'un dossier, afin de ne pas charger toute la base dès le départ.

Bonne pratique associée : **Paginer les longues listes**.

## US05 - Minification du JS et du CSS

En tant qu'utilisateur, je souhaite que le JavaScript et le CSS soient minifiés et débarrassés du code inutile, afin de réduire le poids du code à charger.

Bonne pratique associée : **Minifier et compresser le code**.
